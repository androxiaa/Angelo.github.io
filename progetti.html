Macchina di Turing in Java

Implementazione di una Macchina di Turing utilizzando Java su Visual Studio Code. La Macchina di Turing è un modello di calcolo che manipola simboli su un nastro secondo una tabella di regole. Nonostante la sua semplicità, può essere adattata per simulare la logica di qualsiasi algoritmo del computer.

App per Palestra con Firebase

Un'applicazione mobile per la gestione di una palestra, sviluppata in Java con Android Studio. Utilizza Firebase per implementare funzionalità di login, chat in tempo reale e gestione delle immagini. L'app offre un'interfaccia user-friendly per prenotare lezioni, monitorare i progressi e interagire con gli istruttori.

App per Gestione Collezioni

Un'applicazione Java sviluppata con Android Studio per la gestione di collezioni. Include un sistema di login esclusivo per gli amministratori e un accesso separato per i visitatori. Gli amministratori possono aggiungere, modificare e eliminare elementi della collezione, mentre i visitatori possono visualizzare e cercare all'interno della collezione.

Progetto di Basi di Dati

Un progetto completo di basi di dati che dimostra competenze nella progettazione, implementazione e gestione di database relazionali. Il progetto include la modellazione ER, la normalizzazione, l'implementazione di query SQL complesse e l'ottimizzazione delle prestazioni del database.
